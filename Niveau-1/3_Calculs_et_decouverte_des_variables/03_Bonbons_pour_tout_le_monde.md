# 3. Bonbons pour tout le monde !

*4 classes, constituées respectivement de 25, 30, 27 et 22 élèves. Cependant, 8 élèves sont absents*, le robot doit donner 3 bonbons à chaques élèves, faire l'addition du nombre d'élève moins les 8 abscents :

```Java
class Main {
   public static void main(String[] args) {
      System.out.println((25 + 30 + 27 + 22 - 8) * 3);
   }
}
```
